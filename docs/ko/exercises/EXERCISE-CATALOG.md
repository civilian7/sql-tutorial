# 연습문제 카탈로그 (검토용)

> 이 문서는 실제 배포되지 않습니다. 문제 작성 전 목록을 검토하기 위한 기획 문서입니다.

## 설계 원칙

1. **모든 문제는 TechShop 샘플 DB(ecommerce-ko.db)로 검증 가능**
2. **해당 난이도까지 배운 개념만 사용** (초급 문제에 JOIN 없음 등)
3. **각 문제에 접힌 힌트 + 접힌 정답 + 결과 예시 제공**
4. **개념당 최소 3문제** (기초 → 응용 → 실전)
