# SQL 튜토리얼 - 전자상거래 데이터베이스

현실적인 **전자상거래 데이터베이스**를 사용한 실습형 SQL 튜토리얼입니다. 컴퓨터 및 주변기기를 판매하는 온라인 쇼핑몰의 10년간 비즈니스 데이터를 기반으로 SQL을 학습합니다.

## 학습 과정

| 단계 | 주제 | 레슨 |
|------|------|------|
| **초급** | SELECT, WHERE, ORDER BY, 집계함수, GROUP BY, NULL | 1-6 |
| **중급** | JOIN, 서브쿼리, CASE, 날짜/문자열 함수, DML | 7-14 |
| **고급** | 윈도우 함수, CTE, EXISTS, 뷰, 인덱스, 트리거 | 15-20 |
| **실전 연습** | 비즈니스 문제 해결 | 4개 모듈 |

## 데이터베이스

이 튜토리얼은 가상의 온라인 쇼핑몰 **테크샵(TechShop)**의 데이터베이스를 사용합니다:

- **21개 테이블**: 고객, 상품, 주문, 결제, 배송, 리뷰 등
- **18개 뷰**: 고급 SQL 패턴을 보여주는 사전 구축 쿼리
- **약 23만 건**의 현실적이고 상호 연결된 데이터
- **10년간**의 비즈니스 이력 (2016-2025)

## 사용 방법

1. 샘플 데이터베이스 `tutorial.db`를 SQL 도구에서 열기
2. 각 레슨을 순서대로 따라하기 — 앞 레슨이 다음 레슨의 기초가 됩니다
3. 결과를 보기 전에 **직접 쿼리를 실행**해보세요
4. 각 레슨 마지막의 연습 문제를 풀어보세요
5. 힌트/정답은 막힐 때만 확인하세요

!!! tip "학습 팁"
    쿼리를 읽기만 하지 말고 **직접 타이핑하세요**. SQL도 프로그래밍과 마찬가지로 손으로 익혀야 합니다. 쿼리를 수정해보고, 일부러 오류를 만들어보고, 고쳐보세요. 그게 진짜 학습입니다.
