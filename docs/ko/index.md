# SQL 튜토리얼 <small>v2.0</small>

현실적인 **전자상거래 데이터베이스**로 SQL을 배우는 실습형 튜토리얼입니다.

컴퓨터 및 주변기기를 판매하는 가상의 온라인 쇼핑몰 **테크샵(TechShop)**의 10년간 비즈니스 데이터를 직접 쿼리하며, 기초부터 고급까지 체계적으로 학습합니다.

## 왜 이 교재인가

대부분의 SQL 교재는 문법을 설명하지만, 쿼리를 실행해볼 데이터가 없거나 행이 10~20개에 불과합니다. 실무에서는 여러 데이터베이스를 오가는데, 교재는 하나의 방언만 가르칩니다.

이 튜토리얼은 그 간극을 메우기 위해 만들어졌습니다.

| 기존 교재 | 이 튜토리얼 |
|-----------|------------|
| 문법 설명만, 데이터 없음 | **68만 건**의 현실적 데이터 제공 |
| 10~20행 샘플 데이터 | 10년간의 성장 곡선, 계절성, 고객 행동 패턴 |
| 단일 DB 방언 | **SQLite, MySQL, PostgreSQL** 세 가지 DB별 탭 |
| 영어 또는 한국어 | 데이터와 문서 모두 **한국어/영어** 지원 |

## 무엇을 배우나

22개 레슨, 208개 연습 문제로 구성되어 있습니다. 모든 연습 문제에는 정답과 해설이 포함되어 있습니다.

| 단계 | 주제 | 레슨 | 연습 |
|------|------|:----:|:----:|
| **초급** | SELECT, WHERE, ORDER BY, 집계, GROUP BY, NULL | 6개 | 60문제 |
| **중급** | JOIN, 서브쿼리, CASE, 날짜/문자열, DML, DDL, Self/Cross JOIN | 10개 | 93문제 |
| **고급** | 윈도우 함수, CTE, EXISTS, 뷰, 인덱스, 트리거 | 6개 | 55문제 |

데이터베이스는 30개 테이블과 18개 뷰로 구성되어 있으며, 고객, 상품, 주문, 결제, 배송, 리뷰, 포인트, 프로모션, Q&A 등 쇼핑몰 운영의 전 영역을 다룹니다.

## 시작하기

```bash
pip install -r requirements.txt
python generate.py --size small
```

생성된 `output/ecommerce.db`를 SQL 도구에서 열면 바로 실습을 시작할 수 있습니다.
